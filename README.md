# islands

Calculate the islands from the given matrix of 0s and 1s

## Example of usage:

`bash count_islands.sh "examples/ex1.txt"`

or via Docker (just change the last line in Dockerfile)

`RUN python src/main.py -f "examples/ex1.txt"`

### Python version

Python 3.7.6
